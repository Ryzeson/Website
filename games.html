How to play
Rules:
Birds of a Feather is solitare card game created by Todd Neller. The rules are simple, and although some deals may be challenging, just remember that over 99% of them are solvable!

Controls:
Click on a card and drag it onto the desired stack.
Press the 'u' key to undo a move.

Setup:
You must have Java and JavaFX downloaded on your machine. Run the following command in the same directory that BirdsofaFeather.jar was downloaded to launch the game. (Make sure to replace the module path in the below command with your machine's path to the \lib folder of the javafx-sdk)
java --module-path "\path\to\javafx-sdk-21\lib" --add-modules javafx.controls,javafx.fxml,javafx.base,javafx.graphics,javafx.media,javafx.web,javafx.swing -jar BirdsofaFeather.jar
How to play
Rules:
The game consists of an n by n grid of lights. When the game starts, a random number or a stored pattern of these lights is switched on. Pressing any of the lights will toggle it and the adjacent lights. The goal of the puzzle is to switch all the lights off, preferably with as few button presses as possible. 1

How to play
Rules:
Chomp is a two-player strategy game played on a rectangular grid made up of smaller square cells, which can be thought of as the blocks of a chocolate bar. The players take it in turns to choose one block and "eat it" (remove from the board), together with those that are below it and to its right. The top left block is "poisoned" and the player who eats this loses. 1

Controls:
Click on an empty square to begin the game. Click on any filled cell to 'chomp' it. When the game is over, click any empty square to start a new game.

Setup:
You must have Java and JavaFX downloaded on your machine. Run the following command in the same directory that Chomp.jar was downloaded to launch the game. (Make sure to replace the module path in the below command with your machine's path to the \lib folder of the javafx-sdk)
java --module-path "\path\to\javafx-sdk-21\lib" --add-modules javafx.controls,javafx.fxml,javafx.base,javafx.graphics,javafx.media,javafx.web,javafx.swing -jar Chomp.jar
